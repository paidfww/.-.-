# Жоба туралы
Бұл жоба — "Менің бірінші ЖИ-ассистентім" практикалық жұмысының бөлігі. 
Веб-бет әзірлеушілерге арналған ЖИ құралдары туралы ақпарат береді және Groq API арқылы жұмыс істейтін кірістірілген чат-ботты қамтиды.

## Қалай іске қосу керек
1. Репозиторийді клондаңыз.
2. `script.js` файлын ашып, `GROQ_API_KEY` айнымалысына **Groq API кілтіңізді** жазыңыз.
   - API кілтті [console.groq.com](https://console.groq.com) сайтынан тегін алуға болады.
3. `index.html` файлын браузерде ашыңыз (немесе Live Server қолданыңыз).
4. Төменгі чат арқылы ЖИ-ге сұрақ қойыңыз.

## Пайдаланылған құралдар
- **HTML5**: Семантикалық құрылым.
- **CSS3**: Flexbox, Grid, CSS айнымалылары (Custom Properties), адаптивті дизайн.
- **JavaScript (ES6)**: Intersection Observer API, Fetch API, Groq API интеграциясы.
- **Groq Cloud**: Llama 3 70B моделіне қол жеткізу.

## Мен не үйрендім
- ЖИ API-мен жұмыс істеу үшін `fetch` әдісін қолдану.
- CSS айнымалылары арқылы толыққанды Қараңғы/Ашық тақырыптарды басқару.
- `Intersection Observer` көмегімен элементтердің экранға шығуын бақылау және анимация қосу.
- Заманауи веб-стандарттарды сақтай отырып, ЖИ құралдарын көмекші ретінде пайдалану (кодты түсіну және жақсарту).
